This plugin generates a social media feed based on EDENSSMA.wpengine.com data.

== Description ==

EDENS SMA plugin pulls data directly from EDENSSMA.wpengine.com by using the url placed into the advanced custom field. All functions, fonts, and the gallery are all included inside of this plugin. A new menu option, "Settings" -> "
